# Pedigree Documentation

This directory is the source of truth for the Pedigree product, architecture, delivery, security, GTM, and design artifacts.

## Start Here

- `product/Pedigree_MVP_PRD.md` — MVP scope, epics, user stories, and acceptance criteria.
- `architecture/Pedigree_High_Level_Architecture.md` — system components, data flow, and deployment model.
- `architecture/Pedigree_Data_Model_Schema.md` — Neo4j and PostgreSQL schemas.
- `api/Pedigree_Core_API_Spec.md` plus `../api/openapi.yaml` — REST/API contract.
- `security/Pedigree_Security_Threat_Model.md` — security principles, STRIDE model, and compliance controls.
- `delivery/Pedigree_90Day_Roadmap.md` — Sprint-by-sprint execution plan.
- `design/pedigree_clickable_demo_and_website_design_spec.md` — website and interactive demo design source.

## Directory Map

- `product/` — PRD and launch asset inventory.
- `architecture/` — architecture, data model, and Microsoft integration details.
- `api/` — human-readable API spec.
- `security/` — threat model and security requirements.
- `delivery/` — roadmap, test plan, infra plan, metrics, and PoC playbook.
- `gtm/` — name, vision, positioning, and explainers.
- `design/` — prototype design spec and screenshots.
